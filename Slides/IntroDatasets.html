Dataset Structure

Essential Vocabulary

  • A variable is some characteristic we are interested in that can differ between individuals in a population.

  • A unit of observational (or observation) is a single individual, object, or entity about which we collect data, to describe one or more of its characteristics (variables).

  • An observation may be called a case, subject, or participant.

  • Ideally, each observational unit has a value for each variable.

A dataset is a collection of data values for the observations and variables in a study (e.g., survey or experiment). A common way to arrange data is to put variables in columns and observations in rows. This is what we call tidy data.

Numerical Variables

Variables that can take on a range of numerical values, where it is sensible and meaningful to perform mathematical operations on those values, are numerical or quantitative variables. These often will include units of measure.

A discrete variable is a numerical variable that can only take on a finite or countably infinite number of distinct, separate values. These are often whole numbers or counts.

A continuous variable is a numerical variable that can have any value within a given range. The set of possible values is infinite. These include concepts like time, temperature, speed, or weight, which are limited in decimals only by our ability to measure.

Some variables that use numbers are not truly numerical, such as area codes or social security numbers.

Categorical Variables

At the name suggests, categorical variables are categories into which we group characteristics, or labels for observation units, such as eye color, gender, or state names.

Categories that have a natural ordering, such as the grade levels freshman, sophomore, junior, and senior, are called ordinal.

If categories do not have a natural or meaningful ordering, then we call that a nominal categorical variable. Our IMS2E textbook treats all categorical variables as nominal.

For convenience, we could use numbers to represent categories, such as Democrat = 1, Republican = 2, Independent = 3, but they would still be categorical. We have to be careful of this when we use software for data analysis.

“Levels of Measurement”

Some textbooks refer to a variables “level of measurement” and identify numerical variables as being interval or ratio type.

A ratio variable has a true zero, in the sense that zero means an absence of the characteristic that is being counted or measured. For example, if I have a bank balance of $0, I have no money, or a water level of zero in a reservoir means there is no water.

For a ratio variable, values have the proportions we expect from the real number line. For example, $6.00 is twice as much actual value as $3.00 (the ratio of quantities is 6/3 = 2).

An interval variable has fixed-sized units, but no true zero. For example, 0C is not an absence of temperature (kinetic energy), and 100C is not twice as hot as 50C. However, each 1C added is the same amount of additional energy.

Hierarchy of Values

The levels of measurement framework has an innate hierarchy.

levels: nominal ordinal interval (discrete or continuous) ratio (discrete or continuous)

Older versions of some data analysis programs like SPSS used to use these levels of measurement. Now SPSS uses three levels of measurement: Nominal, Ordinal, and Scale (grouping numerical Interval and Ratio scales together).

R does not make this variable distinction in quite the same way. We will discuss this when we use R to explore data.

Exploring Datasets

Whats in a Dataset?

Before doing any kind of data modeling or inference, you need to examine your dataset and become familiar with its structure and contentsthis is exploratory data analysis (EDA).

John Tukey (19152000), author of the seminal book Exploratory Data Analysis (1977), referred to EDA as numerical, counting, and graphical detective work. He is also credited with introducing the now-familiar terms “bit” and “software” while working with early computers at Bell Labs.

We will use the dplyr and ggplot2 packages for manipulation and visualization. These tools and more are part of the tidyverse.

R has many functions that help us explore datasets and discover their properties/variables.

Datasets in R

A data frame or tibble is a two dimensional structure/array in R. It is a special case of a list where each component (column) has equal length and can contain a different variable type.

The core installation of R (i.e., {base} R) includes many built-in datasets. We can get a list using the data() function.

Entering data() in the Console pane of RStudio will open a new tab/window listing the names of all the built-in datasets.

We can get a list of the datasets found in other packages using data(package = "packagename"). For example, all the data in ggplot2 can be found using data(package = "ggplot2").

Try data(package = .packages(all.available = TRUE)) .

Build from Scratch

Using data.frame()

We can build a dataset from scratch (i.e., from vectors) using the data.frame() function.

Within the data.frame() function, we specify variable name = a vector of values for the variable. All the vectors must be the same length (i.e., same number of cases).

dataset1 <- data.frame(
  ID = 1:5,
  gender = c("male", "male", "female", "female", "nonbinary"),
  height = c(71, 73, 64, 64, 66),
  weight = c(175, 225, 130, 125, 165)
)

Notice the structure of the code, including the indenting and ().

Using tibble()

A tibble is an “upgraded” dataset structure from the tidyverse. However, it works very similarly to data.frame().

dataset2 <- tibble(
  ID = 1:5,
  gender = c("male", "male", "female", "female", "nonbinary"),
  height = c(71, 73, 64, 64, 66),
  weight = c(175, 225, 130, 125, 165)
)

One major difference between data.frame() and tibble() is the way it formats the vectors. The tibble() function generally imposes less structure. For example, it does not turn character data into a factor vector by default.

Example with Missing Data

NA is Rs system character for missing data, similar to how NULL means that there is no vector (vector of length zero).

library(tidyverse)
dataset3 <- tibble(
  ID = c(1, 2, 4, 5, 7),
  sex = c("male", "male", "female", "nonbinary", NA),
  visits = c(1, 2, 4, 12, 2),
  checkup = c(1, 1, 1, 1, 1),
  follow_up = c(NA, 0, 1, 2, 0),
  illness = c(0, 0, 2, 7, 1 ),
  surgery = c(0, 0, 0, 2, 0),
  ER = c(0, 1, 0, 0, NA),
